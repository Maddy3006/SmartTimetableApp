# Smart Timetable App

A Java Swing-based desktop application to generate and manage timetables
with conflict detection.

## Features
- Timetable generation
- Conflict checking
- User-friendly GUI
- Built using Java Swing

## Technologies Used
- Java
- Java Swing
- OOP Concepts

## How to Run
```bash
javac SmartTimetableApp.java
java SmartTimetableApp
